In addition to color calculations, you can use math in SCSS to calculate widths, font-sizes, etc.

You can write SCSS like this: 

.demo-container { 
  width: 100%;
  background: rgba($black, 0.15);
  @include breakpoint(large){
    float: left;
  }
  @include clearfix;
  article {
    font-size: rem-calc($global-font-size + 5px);
    min-height: 20rem;
    @include breakpoint(large){
      float: left;
      width: 590px / 960px * 100%;
      padding: $gutter + $gutter;    
    }
  }
  aside {
    min-height: 10rem;
    @include breakpoint(large){
      float: left;
      width: 330px / 960px * 100%;
      padding: $gutter $gutter * 2.5; 
    }
  }
}
