There are 3 types of bands

Choose the band based upon how much padding is desired above/below the band contents: landing-band, landing-inner-band, landing-band-sides-only. All three band classes include minimal padding to the left and right sides, to prevent content from touching the edge of the window.

landing-band includes the largest amount of padding above and below the content

landing-inner-band includes a smaller amount of padding above and below the content

landing-band-sides-only includes no padding above and below the content

Solid backgrounds

Any of the backgrounds from the standard template can be used, as well as these landing-page-only backgrounds.

Offwhite
Evergreen

Striped backgrounds

Add class background-stripes to any band to overlay the background with a semi-transparent striped pattern.

Or, add class background-stripes-green to any band to overlay the background with a green-striped pattern.

Green with stripe overlay
White with stripe overlay
Primary with stripe overlay
White with green stripe overlay
Offwhite with green stripe overlay

Row Containers

Choose the row container based upon the maximum width that is desired for the contained rows. At screen widths below 1000px, all of these will look the same.

Example of a content contained by row-container-narrow. The max width of the rows in this band is 1000px.

Example of a content contained by a default size row-container. The max width of the rows in this band is 1200px.

Example of a content contained by row-container-wide. The max width of the rows in this band is 1300px.

Example of a content contained by row-container-wide. The max width of the rows in this band is 1500px.
