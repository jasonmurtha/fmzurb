For information about the files listed on this page go to Product Specific Reports File Help.

Estimated REMIC Production
(Monthly) Provides an estimate of REMIC and Strip volume in current coupons.

Recently Priced REMICs
(Hourly) This report details deals that have been priced for settlement.

Reference REMIC Tradable Supply Report
(Monthly) Disclose to the investors how much of an offered class in a Reference REMIC is available for trading.

Section 856 REMIC Asset Percentage Report
(Quarterly) The percentage of REMIC assets that are real estate assets under Section 856 of IRS code.
